# Keyboard Usage Monitor

An educational Python project for **safe keyboard usage monitoring**.

## Overview

This project was built for study purposes in Python and defensive cybersecurity fundamentals.

Its goal is to monitor keyboard events in a **safe and controlled way**.

## Features

- key usage tracking
- date when key was pressed
- JSON output for collected metrics
- simple structure for learning and future improvements

## Technologies

- Python
- pynput
- json
- os
- datetime
